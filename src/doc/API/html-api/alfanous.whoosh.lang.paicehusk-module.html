Module paicehusk

source code

This module contains an object that implements the Paice-Husk stemming algorithm.

If you just want to use the standard Paice-Husk stemming rules, use the module's ``stem()`` function:

   stemmed_word = stem(word)

If you want to use a custom rule set, read the rules into a string where the rules are separated by newlines, and instantiate the object with the string, then use the object's stem method to stem words:

   stemmer = PaiceHuskStemmer(my_rules_string)
   stemmed_word = stemmer.stem(word)
