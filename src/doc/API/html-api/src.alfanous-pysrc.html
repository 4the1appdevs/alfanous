  5  Summary: 
  6  ======== 
  7  Alfanous is a Quranic search engine provides simple and advanced search services in the diverse information of the Holy Quran and is built on the foundations of modern search engines to ensure high stability and high-speed search and we aspire to provide the benefits of facilitating the search for the user like highlight,suggestions ...Etc.. 
  8   
  9  Alfanous is a search engine in the Arabic language necessarily and it provides the morphological analysis , starting from the extraction of roots and other processes different and this will give the user more liberty in the formulation of queries,and also increases the accuracy of the research and its comprehensiveness. 
 10   
 11  Alfanous is a programming interface (API) a tool of developers working on programs related to the Qur'an and want to take advantage of the search services provided by alfanous 
 12   
 13

172  Available fields 
173  ---------------- 
174  Alfanous main index contains lot of info about Qur'an.the search on them is available using fielded search feature: 
175            
176          >>> سورة:الفاتحة  
177   
178  you can use these fields: 
179   - gid : رقم 
180  aya global id 1 to 6236 
181   - sura_id : رقم_السورة 
182  sura id 1 to 114 
183   - aya_id : رقم_الآية 
184  aya id in a specified sura 
185   - aya : آية 
186  aya text in standard script 
187   - aya_ : آية_ 
188  aya text in standard script vocalized 
189   - uth_ : عثماني_ 
190  aya text in othmani script vocalized 
191   - chapter : فصل  
192  the first level of division on subjects 
193   - topic : فرع  
194  the second  level of division on subjects 
195   - subtopic :  باب  
196  the third level of division on subjects 
197   - subject : موضوع 
198  the subject tags 
199   - rub : ربع 
200  Rubu' 
201   - nisf : نصف 
202  Nisf 
203   - hizb : حزب 
204  Hezb 
205   - juz : جزء 
206  Juz' 
207   - page : صفحة 
208  the page id in Mushaf 
209   - manzil : منزل 
210  Manzil 
211   - ruku : ركوع 
212  Ruku' 
213   - sajda : سجدة

244  Use in development 
245  ================== 
246  the API is divided to these main parties: 
247   - TextProcessing_ : contains the linguistic analysers of Quran and Arabic...etc. 
248   - QueryProcessing_ : contains specified query parsers , some linguistic operations and suggestions systems  
249   - ResultsProcessing_ : contains the operations on the results (Scoring,Sorting,Pagination,extending,highlight) 
250   - Crawling_ : its a basic methode to find documents containing quran ayas.needed more development 
251   - related_ :  contains APIs used to import resources from related projects.these projects are : 
252       - PyCorpus_ : to import  morphology xmlfile from quranic corpus project 
253       - Tanzil_ : to import quran from tanzil project  
254       - VerseByVerse_ : to connect the recitations on the site of VersePyVerse project 
255       - ElixirFM_ : to use ElixirFM in python  
256    
257  the API use a *critical* resource which are the index.alfanous project contains some indexes : 
258   - Quran index : its the main index.contains all info about Qur'an: 
259       - Quranic text in different scripts (standard and uthmani) 
260       - Structural infos like hizb,juz',rubu',manzil..etc 
261       - Subjects  
262       - Statistical infos like words number in aya,letters number in aya ... 
263          
264   - Extended index: contains the translations and taffassir
