source code

This module contains the default search query parser.

This uses the excellent Pyparsing module (http://pyparsing.sourceforge.net/) to parse search query strings into nodes from the query module.

This parser handles:

* 'AND', 'OR', 'NOT' * grouping with parentheses * quoted phrase searching * wildcards, e.g. help* * ranges, e.g. [a TO b] * fields, e.g. title:whoosh

This parser was originally based on the searchparser example code available at:

http://pyparsing.wikispaces.com/space/showimage/searchparser.py
