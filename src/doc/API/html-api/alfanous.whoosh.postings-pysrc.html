 17  """ 
 18  This module contains classes for writing and reading postings. 
 19   
 20  The PostIterator interface is the base interface for the two "cursor" 
 21  interfaces (PostingReader and QueryScorer). It defines the basic methods for 
 22  moving through the posting list (e.g. ``reset()``, ``next()``, ``skip_to()``). 
 23   
 24  The PostingReader interface allows reading raw posting information. Individual 
 25  backends must provide a PostingReader implementation that will be returned by 
 26  the backend's :meth:`whoosh.reading.IndexReader.postings` method. PostingReader 
 27  subclasses in this module provide synthetic readers or readers that wrap other 
 28  readers and modify their behavior. 
 29   
 30  The QueryScorer interface allows retrieving and scoring search results. 
 31  QueryScorer objects will be returned by the :meth:`~whoosh.query.Query.scorer` 
 32  method on :class:`whoosh.query.Query` objects. QueryScorer subclasses in this 
 33  module provide synthetic scorers or scorers that wrap other scorers and modify 
 34  their behavior.

147      that appear in the field and any associated value for each term. For 
148      example, if the term vector format is Frequency, the postings for the term

150       
151          [(u"apple", 1), (u"bear", 5), (u"cab", 2)] 
152

155
