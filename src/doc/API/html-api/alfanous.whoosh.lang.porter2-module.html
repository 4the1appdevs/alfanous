Module porter2

source code

An implementation of the Porter2 stemming algorithm. See http://snowball.tartarus.org/algorithms/english/stemmer.html

Adapted from pyporter2 by Michael Dirolf.

This algorithm is more correct but (at least in this implementation) several times slower than the original porter algorithm as implemented in whoosh.lang.porter.
