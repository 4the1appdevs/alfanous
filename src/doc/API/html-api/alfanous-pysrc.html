  5  Summary: 
  6  ======== 
  7  Alfanous is a Quranic search engine provides simple and advanced search services in the diverse information of the Holy Quran and is built on the foundations of modern search engines to ensure high stability and high-speed search and we aspire to provide the benefits of facilitating the search for the user like highlight,suggestions ...Etc.. 
  8   
  9  Alfanous is a search engine in the Arabic language necessarily and it provides the morphological analysis , starting from the extraction of roots and other processes different and this will give the user more liberty in the formulation of queries,and also increases the accuracy of the research and its comprehensiveness. 
 10   
 11  Alfanous is a programming interface (API) a tool of developers working on programs related to the Qur'an and want to take advantage of the search services provided by alfanous 
 12   
 13   
 14  Features: 
 15  ========= 
 16    - Advanced search features 
 17       - Fielded Search  
 18       - jokers (regular expression)  
 19       - logical relations 
 20       - Phrase Search 
 21       - Intervals search  
 22    - Additional Features 
 23       - spelling Suggestions 
 24       - pagination 
 25       - Sorting 
 26       - Highlight 
 27    - Language options  
 28       - Signification 
 29       - Synonyms and Antonyms (to do) 
 30       - Diacritics  
 31       - Affixes  
 32       - Arabic word Root  
 33       - Verb Conjugaisons 
 34       - Nouns derivations 
 35       - Arabic word type  
 36       - Arabic word Pattern (to do) 
 37       - Spell Errors :hamza formes,some similar letters 
 38    - Options Qur'anic 
 39       - Search in Uthmani script  
 40       - Extend the search to all qur'anic sciences (to do) 
 41       - the ability to search using special indexes  
 42       - Structural options  
 43       - Statistical Informations

 50  Install 
 51  ------- 
 52  use easy_install to download and install alfanous API

 55  Examples 
 56  -------- 
 57  to call the api from script 
 58    - import and call it  
 59           
 60          >>> from alfanous.main import QuranicSearchEngine 
 61          >>> QSE = QuranicSearchEngine()  
 62          <__main__.QuranicSearchEngine instance at 0xb77d168c> 
 63           
 64       
 65    - find the results of a query 
 66               
 67              >>> results,_ =QSE.search_all(u"الحمد") 
 68              >>> for r in results: 
 69              >>>       print r["aya"] 
 70              الحمد لله رب العالمين 
 71       
 72    - show more information about a result : sura's name, aya's number ,aya's vocalized text 
 73               
 74              >>>  for r in results: 
 75              >>>        print r["sura_name"],":",r["aya_id"],":",r["aya_"] 
 76              الفاتحة:2:الْحَمْدُ لِلَّهِ رَبِّ الْعَالَمِينَ          
 77                          
 78    - find the terms used to search from a query 
 79               
 80              >>> _,terms =QSE.search_all(u"الحمد") 
 81              >>> print terms 
 82              [("aya",u"الحمد")] 
 83              >>> print ",".join([term[1] for term in list(terms)]) 
 84              الحمد 
 85       
 86    - use the terms to highlight the aya text 
 87               
 88              >>> Qhighlight(u"الْحَمْدُ لِلَّهِ رَبِّ الْعَالَمِينَ", [u"الحمد", u"لله"]) 
 89              <span style="color:red;font-size:100.0%"><b>الْحَمْدُ</b></span> <span style="color:green;font-size:100.0%"><b>لِلَّـهِ</b></span> رَبِّ الْعَالَمِينَ

 91    - get the search time and the number of results 
 92               
 93              >>> print "time=%f,number=%d" % (results.runtime,len(results)) 
 94              time=0.100318,number=160

 96    - limit the number of results  
 97              
 98              >>> results,_ =QSE.search_all(u"الله",limit=10)
